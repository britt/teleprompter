Class PromptsDurableObject

PromptsDurableObject is a Durable Object that stores prompts. Prompts are versioned and append only.

This object can:

  • return a list of all prompts
  • get a prompt by id
  • get all versions of a prompt by id
  • write a prompt
  • delete a prompt
